Usage Instructions:

1. Start the server:
   go run cmd/minirag/main.go

2. Available endpoints:
   - /chat - Chat with the system
   - /query - Search documents
   - /debug/db - View database state

3. Command line options:
   - --docs - Document directory
   - --data - Data directory
   - --ollama-url - Ollama API URL
   - --ollama-model - Chat model
   - --force-reindex - Force reindexing

The system supports both development and production modes.
